# jm-photo

> 一个用于**分割与重组** jmcomic 漫画图片的网页工具

## 项目简介

`jm-photo` 是一个轻量级的前端网页应用，主要解决 jmcomic 平台上部分漫画图片被“置乱/分割”后难以直接查看的问题。它能够将分割后的图片块按照正确的顺序重新拼接，还原完整画面。

- **核心功能**：上传或指定被分割的漫画图片，通过算法自动计算分割层数，并在网页端完成重组与显示。
- **适用场景**：适用于研究图片反置乱技术、批量浏览本地分割漫画，以及学习前端图片处理与JS逆向分析。
- **项目来源**：本项目基于CSDN博主“糕手”([文章链接](https://blog.csdn.net/qq_36684855/article/details/121582883))与“error=Nil”([文章链接](https://blog.csdn.net/m0_50233241/article/details/148189510))的技术思路开发，测试网站：[https://jm.xiaoli.icpx.store/](https://jm.xiaoli.icpx.store/)

## 已知限制 ⚠️

> **注意**：受制于不同漫画图片的原始分割参数差异，本工具**并非对所有漫画图片都有效**。若遇到无法正确分割或重组的情况，请参考上述两位博主的文章进行代码适配或参数调整。
